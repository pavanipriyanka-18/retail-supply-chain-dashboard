## Retail Supply Chain Analytics Dashboard

**Tools:** SQL, Power BI, Tableau, Excel

### Overview
End-to-end analytics dashboard analyzing 100,000+ inventory 
records to improve supply chain visibility by 40%.

### Key Steps
- SQL queries to extract and transform inventory data
- KPI metric design and dashboard development in Power BI
- Trend and bottleneck analysis across distribution data
- Presented insights to stakeholders for operational decisions

### Results
- 40% improvement in supply chain visibility
- Identified key distribution bottlenecks affecting product availability
